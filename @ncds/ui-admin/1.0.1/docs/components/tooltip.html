Tooltip

툴팁은 요소에 대한 추가 정보를 표시하는 작은 팝업 창입니다. 사용자가 요소에 마우스를 올리거나 포커스할 때 나타납니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 Tooltip 컴포넌트의 모습을 확인해보세요.

툴팁 제목툴팁 내용입니다

툴팁의 제목
contentstring
툴팁에 표시될 내용
positionselect
툴팁이 표시될 위치
hideArrowboolean
툴팁 화살표 숨김 여부
iconTypeselect
툴팁 아이콘 타입
tooltipTypeselect

주의사항​

  1. 툴팁은 viewport를 벗어나지 않도록 자동으로 위치가 조정됩니다.
  2. 툴팁의 내용이 HTML을 포함합니다.
  3. 툴팁은 기본적으로 마우스 호버 시 표시됩니다.

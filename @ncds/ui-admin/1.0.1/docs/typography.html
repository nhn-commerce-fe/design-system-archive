Typography

NCDS Admin은 일관된 타이포그래피 스타일을 제공합니다. 사용한 폰트 크기와 두께는 CSS 변수로 정의되어 있으며, 다양한 상황에서 사용할 수 있습니다. NCDS Admin의 기본 폰트는 다음과 같습니다.

커머스 산스(Commerce Sans)​

:root {
  font-family: 'CommerceSans', sans-serif;
}

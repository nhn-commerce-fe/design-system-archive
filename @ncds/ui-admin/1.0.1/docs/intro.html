NCDS Admin 소개

NCDS Admin은 NHN Commerce의 관리자 페이지를 위한 디자인 시스템입니다. 리액트와 HTML을 기반으로 만든 프론트엔드 툴킷으로, 아토믹 디자인 원칙을 적용해 컴포넌트를 구성했습니다. 이를 통해 일관된 디자인과 효율적인 개발 경험을 제공합니다.

시작하기

현재 버전은 v1.0.0입니다.

시작하기​

NCDS Admin으로 바로 개발을 시작할 수 있습니다. CDN 방식, 패키지 매니저 설치 중 편한 방법을 선택하세요.

설치 방법 알아보기

npm/yarn으로 설치하기​

npm이나 yarn을 통해 NCDS Admin의 소스 파일을 설치할 수 있습니다.
그리고 각 컴포넌트에 있는 데모를 참고해 빠르게 프로젝트를 시작할 수 있습니다.

npm install @ncds/ui-admin

더 자세한 설치 방법이 궁금하다면 설치 가이드를 확인해보세요.

CDN으로 시작하기​

CSS나 JS 파일만 필요하다면 CDN을 활용해보세요.
빠른 시작 가이드를 통해 기본 사용법을 익히고, 예제를 살펴보면서 프로젝트에 적용할 수 있습니다.

<link rel="stylesheet" href="https://fe-sdk.cdn-nhncommerce.com/@ncds/ui-admin/1.0/main.min.css" />
<script src="https://fe-sdk.cdn-nhncommerce.com/@ncds/ui-admin/1.0/main.min.js"></script>

주요 특징​

  • 다양한 컴포넌트: 버튼, 입력 필드, 선택 상자 등 필수적인 UI 요소를 모두 담았습니다
  • 편리한 유틸리티: 대부분의 컴포넌트는 다양한 사이즈를 지원합니다
  • 반응형 지원: 모바일부터 데스크톱까지, 다양한 화면에서 완벽하게 작동합니다

빠른 시작 가이드​

NCDS Admin을 시작하기 위한 세 가지 단계를 소개합니다

  1. 시작하기 문서를 보고 기본 설정을 완료하세요
  2. 필요한 컴포넌트와 유틸리티를 찾아서 적용해보세요
  3. 각 컴포넌트의 속성과 이벤트를 활용해 원하는 기능을 구현하세요

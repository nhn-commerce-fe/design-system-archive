Quick Start

StepGuide는 웹 페이지에서 사용자에게 단계별 가이드를 제공하는 컴포넌트입니다. 이 페이지에서는 StepGuide를 빠르게 시작하는 방법을 설명합니다.

기본 사용법​

StepGuide를 사용하려면 HTML 페이지에 JavaScript와 CSS 파일을 추가해야 합니다. 다음은 기본적인 사용 예시입니다:

<link rel="stylesheet" href="https://fe-sdk.cdn-nhncommerce.com/@ncds/step-guide/1.0/step-guide.min.css" />
<script src="https://fe-sdk.cdn-nhncommerce.com/@ncds/step-guide/1.0/step-guide.min.js"></script>

그런 다음, JavaScript에서 StepGuide를 초기화하고 사용할 수 있습니다:

const guide = stepGuide({
  el: document.body,
  options: {
    steps: [
      {
        element: '#first-step',
        title: '첫 번째 단계',
        description: '이것은 첫 번째 단계입니다.',
      },
    ],
  },
});

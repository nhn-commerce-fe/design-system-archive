ProgressCircle

프로그레스 서클은 진행 상태를 원형 그래프로 표시하는 컴포넌트입니다. 특정 작업의 완료율이나 목표 달성도를 시각적으로 표현하는 데 적합합니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 프로그레스 서클의 모습을 확인해보세요.

기본 사용법​

import { ProgressCircle } from '@ncds/ui-admin';

function App() {
  return <ProgressCircle size="md" value={40} />;
}

Props​

size'xxs' | 'xs' | 'sm' | 'md' | 'lg'-프로그레스 서클의 크기
valuenumber-진행률 값 (0-100)
labelstring진행률(%)표시할 레이블 텍스트
theme'circle' | 'half-circle''circle'서클 모양 (원형 또는 반원형)

크기​

ProgressCircle은 5가지 크기를 지원합니다: xxs, xs, sm, md, lg

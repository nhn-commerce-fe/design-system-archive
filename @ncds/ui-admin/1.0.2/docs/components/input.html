Input

Input 컴포넌트는 사용자로부터 텍스트 데이터를 입력받기 위한 UI 요소입니다. 이 컴포넌트는 다양한 사이즈와 상태를 지원하며, 아이콘, 텍스트, 버튼 등의 요소를 앞뒤에 배치할 수 있습니다.

function App() {
  return <InputBase placeholder="텍스트를 입력하세요" />;
}
